# HR PDF to Text - n8n Workflow
Ce workflow télécharge un CV depuis Google Drive, convertit le PDF en texte, puis analyse le CV avec un modèle multimodal Google Gemini pour déterminer si le candidat est qualifié.

## Comment l'utiliser
1. Importer `workflow.json` dans votre instance n8n.
2. Configurer vos clés API Google et Stirling PDF.
3. Exécuter le workflow.
